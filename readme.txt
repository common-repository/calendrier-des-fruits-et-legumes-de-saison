== Description==

Pour avoir le meme calendrier illustre des fruits et legumes de saison que sur http://fr.gaiadir.com/ sur votre blog wordpress
    
== Installation ==

1. Decompresser le fichier *.zip vers le dossier `/calendrier-produit-saison/`.
2. Uploader le dossier `/calendrier-produit-saison/` vers le dossier `/wp-content/plugins/` de votre hebergement de wordpress par ftp
3. Alers dans Extension > Installees et activer le plugin
4. Allers dans Apparence > Widgets, et ajoute `Calendrier produit de saison` a votre colonne laterale
5. Changer le titre du widget si necessaire et sauvegarder.
